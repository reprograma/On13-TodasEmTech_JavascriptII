Começaremos esse artigo explicando um pouco do poder da linguagem de programação web mais popular em todo o mundo, o todo poderoso JavaScript. As páginas web já tiveram outra cara, com pouco ou quase nenhuma interatividade e dinamismo, o que não tornava as coisas tão atrativas e limitavam o funcionamento geral dos sites. Com o surgimento das linguagens de script e posteriormente a popularização do JavaScript, tudo mudou, foi um BANG! para os desenvolvedores e para o futuro, proporcionando um dinamismo e uma maior interação com o usuário entre outras diversas funcionalidades que veremos durante o decorrer do artigo.
